# Markdown2Openxml.NET
Convert Markdown string to Openxml format for Word Document

# Currently the library supports the following format:
## Heading
```
# Heading1
## Heading2
### Heading3
```

## Bold
```
**Bold**
```

## Italic
```
*Italic*
```

## Code Block
```
\\```
Code block
\\```
```

## Table
```
| | |
|---|---|
| | |
```

## Link
```
[Name](http://url)
```

## Image
```
![Attachment](data:image/png;base64,images-hash)
```

## Bullet list
```
- Bullet list
```

## Ordered List
```
1. Order1
2. Order2
```

## Orizontal Rule
```
* * *
```
